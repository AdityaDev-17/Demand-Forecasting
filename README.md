# 📌 Project Overview

Demand forecasting plays a critical role in inventory management, supply chain optimization, and business planning. Accurate forecasts help organizations:

* Reduce inventory costs
* Prevent stockouts
* Improve resource allocation
* Support data-driven decision-making

This project uses historical sales data and forecasting models to predict future demand and visualize trends through an interactive Streamlit application.

---

# 🎯 Business Problem

Businesses often struggle with:

* Overstocking inventory
* Underestimating demand
* Inefficient supply chain planning
* Revenue loss due to poor forecasting

The objective of this project is to build a forecasting system that can predict future demand patterns and help organizations make informed decisions.

# 🤖 Forecasting Models

## Prophet

Prophet is a time series forecasting model that:

* Handles seasonality and trends
* Works well with business data
* Handles missing values effectively
* Produces interpretable forecasts

---

## N-Beats

N-Beats is a deep learning forecasting architecture that:

* Learns complex temporal patterns
* Handles non-linear relationships
* Produces highly accurate forecasts
* Works well on demand forecasting problems
